# Intelligent Resume Screening System

## Overview
An AI-based system that automates resume screening by analyzing, classifying, and ranking resumes based on job requirements using NLP and Machine Learning.

## Features
- Resume Text Extraction and Preprocessing
- NLP-based Text Cleaning using NLTK
- TF-IDF Feature Extraction
- Resume Classification using Machine Learning
- Candidate Ranking using Cosine Similarity
- Automated Resume Shortlisting

## Technologies Used
- Python
- NLTK
- Scikit-learn
- Pandas
- NumPy
